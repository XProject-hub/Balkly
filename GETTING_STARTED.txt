UBUNTU 22 VPS DEPLOYMENT:
--------------------------
1. Install Docker:
   curl -fsSL https://get.docker.com | bash

2. Clone and setup:
   git clone <repo> /var/www/balkly
   cd /var/www/balkly
   ./setup.sh

3. Configure:
   nano balkly-api/.env
   # Add Stripe keys, email SMTP

4. Access:
   http://your-server-ip
