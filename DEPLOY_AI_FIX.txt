========================================
DEPLOY AI HELPER - OpenAI Integration
========================================

NA VPS-u uradi ovo:

1. Pull latest code:
   cd /var/www/balkly
   git pull origin main

2. Dodaj OpenAI API key u .env:
   nano balkly-api/.env
   
   Dodaj na kraj fajla:
   
   OPENAI_API_KEY=<your_openai_api_key>
   OPENAI_MODEL=gpt-4o-mini
   
   (API key je u Slack/Email - ne čuvaj ga u Git!)
   
   Sačuvaj (Ctrl+O, Enter, Ctrl+X)

3. Restart backend:
   docker-compose restart api
   sleep 5

4. Clear cache:
   docker exec balkly_api php artisan config:clear
   docker exec balkly_api php artisan cache:clear

5. Test:
   - Idi na https://balkly.live/listings/create
   - Unesi naslov i opis
   - Klikni "Auto-Enhance Listing"
   - Treba da AI poboljša tekst BEZ EMOJIS!
